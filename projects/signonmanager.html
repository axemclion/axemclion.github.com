Sign On Manager
Even though passwords are saved by the browser, we have to press the login button. This is a project that experiemtns with ways to press the "Login" button automatically.
Website http://axemclion.github.com/googlepages/SignOnManager.html/
Project News http://dy-verse.blogspot.com/search/label/signonmanager
SignOn Manager lets you log in to websites automatically. It works both with Form Submit and Javascript links This page is a demonstration of how the Sign On Manager Works. This page uses a script to simulate Firefox GreaseMonkey Plugin. Steps to use the Script Demo
  1. Enter any user name password in the form to the right
  2. Before hitting the submit button, click the SignOn button in brown Div at the top center
  3. The next time you access this page, username/password is entered and form submitted
You can also use the GMSimulator window to see the various values stored in the about:conig The TOOLS > GreaseMonkey > UserScripts Menu is also shown in the Simulator.
