Bookmark to delicious
A ubiquity command that automatically generates tags for a page and bookmarks the link on Delicious, with selected text as notes.
Source Code http://gist.github.com/26425
Project News http://dy-verse.blogspot.com/search/label/ubiquity%3Abookmark
Click here to subscribe to the feed. If you want to bookmark a page you are browsing, simply select the text you want to be saved as notes and use "Bookmark" ubiquity command. Tags are automatically generated and the page is bookmarked on your delicious account.
If you are not logged in, your credentials will be asked for, as this is over HTTP Authentication.
