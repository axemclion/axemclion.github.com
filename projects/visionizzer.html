Visionizzer
A code generation tool to create GUI for DOS based C programs
Source Code http://code.google.com/p/visionizzer/0
Blog http://dy-verse.blogspot.com/search/label/visionizzer
Downloads http://code.google.com/p/visionizzer/downloads/list
This is a C/C++ code generation tool that give you visual basic style GUI designing options and generates code for the UI designed. It has the ability to add plugins to draw various other interfaces (like electronics, etc). Some sample applications developed with this GUI are also hosted in the branches. This project has also undergone name changes and was called Draw, the version developed in GW Basic Visionizzer - versions 2 to 5 Shrishti - Version 6 that was a combination of all applications.
This project was a hobby project at college and is no longer in active development. It is hosted on Google code as an archive. If you want to use the code, please contact the project author.
