Linkify
A ubiquity command to search and create links out of words in a web page.
Source Code http://gist.github.com/28928
Project News http://dy-verse.blogspot.com/search/label/linkify
Click here to subscribe to the feed. Steps to use this ubiquity command.
  1. Select text in the web page.
  2. Invoke Ubiquity and type in "linkify".
  3. A preview pane with YAHOO BOSS search results is displayed.
  4. To convert the selection to a link, click the search result.
  5. The text you selected becomes linkified.
