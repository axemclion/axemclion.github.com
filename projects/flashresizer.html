Flash Resizer :: Bookmarklet
A bookmark to resize/move movies, games and other flash content on web pages to see them better.
Source Code http://gist.github.com/89202
Blog http://dy-verse.blogspot.com/search/label/flash%20resizer
Bookmarklet Flash Resizer (drag and drop this link in bookmarks toolbar)
Flash content on many sites can be seen better if they could be resized to larger dimensions. Sites like youtube and vimeo do allow full screen videos but that usually hides the whole page.
With this bookmarklet, you can select any flash content, move it and re-size it by dragging the corners. Add this FLASH RESIZER link to your bookmarks or drag and drop it to the bookmarks toolbar.
