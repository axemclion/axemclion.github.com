Seam Carving Demo
Implementation of the Seam Carving algorithm in Javascript using the Canvas Tag
Project News http://dy-verse.blogspot.com/search/label/seam%20carving
Website http://axemclion.github.com/seamcarving/
Seam Carving is a technique to resize images without distorting its proportions. More information about this can be found here and here.
This project implements the vertical seam carving algorithm in Javascript using the HTML 5 Canvas tag. Please note that the exampl may require Firefox, Google Chrome or a browser that supports HTML 5 Canvas Tag.
