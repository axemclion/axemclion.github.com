ScrapsTimeout
Place hidden gifts that open at the specified times and send them as emails or Orkut scraps.
Source Code http://github.com/axemclion/scrapstimeout
Project News http://dy-verse.blogspot.com/search/label/scrapstimeout
Web site http://axemclion.github.com/scrapstimeout/
Send gifts as orkut scraps. Gifts can be hidden inside a gift wrapper that can be opened only at a specified time. Just select the gift, type the message and set the time.
The viewer will have to wait till the time before the gift can be opened. Add a suspense element to gifts now !!
