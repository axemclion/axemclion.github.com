Map Doodle
A ubiquity command to search and create links out of words in a web page.
Website http://axemclion.github.com/googlepages/mapdoodle.html
Project News http://dy-verse.blogspot.com/2007/10/hack-doodle.html
The application is a website with Yahoo Maps on which you could draw a path. This drawing sequence is recorded and can be shared with friends for replay.
This was a project put together in less than 24 hours at the Yahoo Hack Day, India and was awarded the "Best in Show" award.
